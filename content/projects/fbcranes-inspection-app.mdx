## Project Overview

This mobile application extends FB Cranes' inspection management system to the field:

- **NFC Scanning**: Quick access to equipment data via NFC tags
- **Serial Search**: Efficient equipment lookup by serial number
- **Real-time Data**: Instant access to latest inspection records
- **Offline Support**: Functionality maintained without connectivity
- **Cross-platform**: iOS and Android support

## Technical Implementation

### Mobile Architecture
- **Expo**: Cross-platform development framework
- **NativeWind**: Tailwind CSS for React Native
- **TypeScript**: Type-safe development
- **React Navigation**: Seamless navigation
- **Expo Router**: File-based routing

### NFC Integration
- **React Native NFC**: Native NFC capabilities
- **Tag Reading**: NDEF message parsing
- **Error Handling**: Robust NFC error management
- **Platform Support**: iOS and Android compatibility
- **Background Scanning**: Efficient tag detection

### Data Management
- **React Query**: Efficient data fetching
- **Offline Storage**: AsyncStorage integration
- **State Management**: Zustand for global state
- **Data Sync**: Background synchronization
- **Caching**: Smart data caching

### UI/UX Features
- **NativeWind**: Tailwind-style styling
- **Native Components**: Platform-specific design
- **Animations**: React Native Reanimated
- **Haptic Feedback**: Enhanced user interaction
- **Dark Mode**: Automatic theme switching

### API Integration
- **REST Client**: Axios integration
- **WebSocket**: Real-time updates
- **JWT Auth**: Secure authentication
- **Request Caching**: Optimized API calls
- **Error Handling**: Graceful failure recovery
