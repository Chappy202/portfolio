# PartsCorp - High-Performance Ecommerce Platform

PartsCorp is a high-performance ecommerce platform designed for handling large product catalogs efficiently. Built on MedusaJS V1, it provides a seamless shopping experience for both B2B and B2C customers in the automotive and industrial parts sector.

## Core Features

- **Near-Instant Search**: Integrated with Algolia for lightning-fast product search and discovery
- **Multi-Tenant Architecture**: Support for multiple vendors and storefronts
- **B2B & B2C Support**: Tailored experiences for both business and consumer customers
- **Multi-Region Capabilities**: Region-specific pricing, shipping, and tax handling
- **SEO Optimized**: Built with best practices for search engine visibility
- **High Performance**: Optimized loading times for large product catalogs

## Technology Stack

The platform leverages modern technologies for optimal performance and scalability:

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: MedusaJS, Node.js
- **Database**: PostgreSQL
- **Search**: Algolia
- **Deployment**: Vercel
- **Performance Monitoring**: Vercel Analytics
