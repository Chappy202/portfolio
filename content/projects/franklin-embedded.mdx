# Franklin Electric Embedded Controller Interface

A highly optimized web interface for Franklin Electric's embedded pump controllers, providing real-time monitoring, control, and diagnostic capabilities through a responsive and reliable platform.

## Project Overview

This application serves as the primary interface for Franklin Electric's embedded pump controllers, offering:

- **Real-time Monitoring**: Live data streaming through SSE
- **Advanced Error Handling**: Sophisticated recovery mechanisms
- **Optimized Bundle Size**: Minimal footprint for embedded systems
- **PWA Support**: Offline capabilities and mobile optimization
- **Comprehensive API Integration**: Custom caching and state management
